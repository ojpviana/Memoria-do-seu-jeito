Pasta dos codigos de banco de dados que estão em testes para implementação.

Para utilizar o banco de dados é necessario os arquivos "package-lock.json" e "package.json":
No terminal, inicie e instale o sqlite utilizando os comandos abaixo na mesma ordem,
  1 - npm init -y
  2 - npm install sqlite
  3 - npm install sqlite3

Comandos para verificação do sqlite,
  - npm list sqlite
  - npm list sqlite3

Comando para rodar o banco de dados,
  - node ./app.js
